# 바이브코딩으로 1시간 만에 AI 서비스 런칭하기 | 디자인부터 결제, 배포까지

## 1. Summary
이 강의는 바이브코딩과 AI 도구를 활용하여 유튜브 썸네일 자동 생성 서비스를 1시간 내에 디자인, 개발, 결제 시스템 구축, 배포까지 완성하는 과정을 단계별로 안내한다. 프로젝트 설정, Next.js 환경 구축, 슈퍼베이스 연동, AI 이미지 생성, 결제 기능 구현, 크레딧 관리, UI/UX 개선 및 배포까지 실무적으로 필요한 핵심 기술들을 다룬다.

## 2. Key Points
- 명확한 기획 없이는 AI 자동화 프로젝트가 방향성을 잃기 쉬움.
- Next.js, GitHub, AI 코드 보조 도구를 활용해 빠르게 개발 환경 구성.
- 슈퍼베이스를 통한 인증 및 사용자 관리(구글 로그인) 구현.
- 제미나이 API를 이용해 AI 썸네일 이미지 생성 기능 개발.
- 폴라(Pola) 결제 시스템 통합으로 글로벌 구독·결제 처리 자동화.
- 구독 상태에 따른 크레딧 로직 및 UI 실시간 반영.
- 버셀을 활용한 간단한 배포 및 도메인 연결 방법.
- 완성된 서비스는 SEO, 보안, 에러 추적, 마케팅과 같은 확장 가능성이 있음.

## 3. Concepts
- **바이브코딩**: AI를 활용하여 개발자가 효율적으로 코드를 작성하고 서비스 개발을 자동화하는 방법론.
- **Next.js**: React 기반의 서버 사이드 렌더링과 정적 사이트 생성 기능을 지원하는 프레임워크.
- **슈퍼베이스(Supabase)**: 오픈소스 백엔드 서비스 플랫폼으로, 데이터베이스, 인증, 스토리지 기능을 제공.
- **폴라(Pola)**: 글로벌 구독 결제 솔루션으로, 사업자 없이도 결제 및 세금 처리를 지원.
- **환경 변수(Env Variables)**: API 키 등 민감한 정보를 코드에 직접 저장하지 않고 별도로 관리하는 방식.
- **외폭(Webhook)**: 외부 서비스에서 주기적 또는 이벤트 기반으로 특정 URL에 알림을 보내는 기술.
- **크레딧 로직**: 사용자가 결제를 통해 얻은 사용 가능 포인트로 AI 이미지 생성 등 서비스 이용 권한을 관리하는 시스템.
- **버셀(Vercel)**: Next.js 개발사에서 만든 무중단 배포 플랫폼으로 간편하게 웹 애플리케이션을 호스팅할 수 있음.

## 4. Study Plan
- 1일차: 프로젝트 기획과 Next.js 환경 설정 및 GitHub 연동 익히기
- 2일차: 슈퍼베이스로 사용자 인증(구글 로그인)과 데이터베이스 구축 이해하기
- 3일차: AI 이미지 생성 API(제미나이) 연동 및 프롬프트 엔지니어링 학습
- 4일차: 결제 시스템 폴라 통합 및 외폭(Webhook) 이벤트 처리 방법 익히기
- 5일차: 크레딧 관리 로직 및 UI 구현과 모바일 최적화 방법 실습
- 6일차: 배포, 도메인 연결, 버셀 환경 변수 설정 및 배포 점검
- 7일차: 전체 서비스 흐름 리뷰 및 향후 확장 가능한 서비스 구조 이해
- 8일차: 프로젝트 코드를 재검토하고 문제 해결과 최적화 연습
