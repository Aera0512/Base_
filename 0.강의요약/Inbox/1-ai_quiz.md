# Quiz - 바이브코딩으로 1시간 만에 AI 서비스 런칭하기 | 디자인부터 결제, 배포까지

## Q1 (multiple_choice)
바이브코딩에서 가장 중요한 초기 단계는 무엇인가요?
- 코드 작성
- 기획 및 목표 설정
- 디자인 완성
- API 키 발급

### Answer
기획 및 목표 설정

### Explanation
기획이 명확해야 AI 자동화 프로젝트가 올바른 방향으로 진행되고 개발 과정에서 불필요한 시간 낭비를 줄일 수 있다.

## Q2 (multiple_choice)
Next.js에서 페이지를 구성할 때 권장되는 방식은 무엇인가요?
- 모든 코드를 하나의 큰 파일에 작성한다
- 페이지 코드는 작게 유지하고 컴포넌트로 분리한다
- HTML 파일을 별도로 작성한다
- 코드를 전부 서버에 두고 클라이언트에는 보내지 않는다

### Answer
페이지 코드는 작게 유지하고 컴포넌트로 분리한다

### Explanation
유지보수와 확장성을 위해서 페이지별 코드는 최소화하고 여러 컴포넌트로 구조화 하는 것이 좋다.

## Q3 (true_false)
슈퍼베이스 인증 기능에서는 구글 로그인을 위한 클라이언트 아이디와 시크릿 키가 필요하다.
- true
- false

### Answer
true

### Explanation
구글 로그인 연동을 위해서는 구글 클라우드 콘솔에서 클라이언트 아이디와 시크릿 키를 생성하여 슈퍼베이스에 등록해야 한다.

## Q4 (short_answer)
글로벌 구독 결제 시스템으로 많이 사용하는 서비스 중 한국 사업자가 아닌 경우 대안으로 사용되는 서비스는 무엇인가요?

### Answer
폴라(Pola)

### Explanation
폴라는 별도의 해외 법인 없이 글로벌 결제가 가능하도록 지원하는 서비스로, 한국 사업자가 스트라이프를 쓸 수 없을 때 대안이 된다.

## Q5 (multiple_choice)
환경 변수 파일(.env)에 저장된 서비스 롤키(Service Role Key)는 어디서만 사용해야 하며 이유는 무엇인가요?
- 프론트엔드에서 사용해도 무관하다
- 백엔드에서만 사용해야 하며 노출 방지를 위해서다
- 배포 서버의 public 폴더에 저장한다
- 코드 저장소에 그대로 커밋한다

### Answer
백엔드에서만 사용해야 하며 노출 방지를 위해서다

### Explanation
서비스 롤키가 노출되면 모든 데이터에 접근 권한이 생기므로 백엔드에서만 보안에 유의하며 사용해야 한다.

## Q6 (true_false)
엔그록을 사용하면 로컬에서 실행 중인 서버를 임시 공용 주소로 노출시킬 수 있다.
- true
- false

### Answer
true

### Explanation
엔그록은 로컬 환경에 외부에서 접속할 수 있도록 터널링 서비스를 제공한다.

## Q7 (multiple_choice)
크레딧 관리 시스템에서 크레딧이 부족할 때 일반적인 UI 반응은 무엇인가요?
- 에러 메시지만 출력한다
- 자동으로 크레딧 구매 모달을 보여준다
- 서비스를 강제로 종료한다
- 크레딧을 무료로 추가한다

### Answer
자동으로 크레딧 구매 모달을 보여준다

### Explanation
사용자 경험을 위해 크레딧 부족 시 바로 구매를 유도하는 모달을 띄워 서비스 이용을 원활히 한다.

## Q8 (short_answer)
Next.js 공식 배포 서비스 이름은 무엇인가요?

### Answer
버셀(Vercel)

### Explanation
Next.js를 만든 회사가 운영하는 클라우드 플랫폼으로, 손쉽게 배포 및 환경 변수 설정이 가능하다.
