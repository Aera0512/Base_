# 영어 학습 몰입을 위한 뇌과학 적용 가이드

> [!abstract] 핵심 원칙
> 영어 학습 효율은 **의지력이 아니라 뇌의 상태**가 결정한다.
> 시간대별 뇌내물질 흐름에 맞춰 학습을 배치하면 같은 시간으로 더 많이 흡수할 수 있다.

---

## 핵심 뇌내물질 4가지

### 1. 세로토닌 — 오전 학습의 기반

**역할:** 새로운 개념 학습, 논리적 이해(RC 문법/구문), 집중력 있는 딥러닝

**분비 조건:**
- 기상 후 30분 이내 햇빛 노출 (15-30분)
- 규칙적 기상 시간
- 트립토판 음식 (계란, 바나나, 견과류)

**영어 학습 적용:**
- 가장 어려운 학습(새 단어, RC 정독, 문법)을 기상 후 2-3시간 내에
- 오전에 전략적 사고가 필요한 학습 배치

---

### 2. 도파민 — 오후 실행 & 반복 학습

**역할:** 목표 추구, 반복 문제풀이, 체크리스트 기반 학습 동기 부여

**분비 조건:**
- 구체적 목표 설정 + 달성
- 작은 성취의 시각화 (체크리스트)
- 완료 후 작은 보상

**영어 학습 적용:**
- 토익 LC/RC 문제풀이는 오후에
- 포모도로 25분 단위로 쪼개서 "1세트 완료" 표시
- 완료 후 작은 보상 (커피, 좋아하는 음악 1곡)

---

### 3. 아세틸콜린 — 저녁 스피킹 & 창의적 출력

**역할:** 능동적 출력, 통찰, 연결 짓기, 스피킹 연습

**분비 조건:**
- 이완 상태 (산책, 샤워 후)
- 충분한 수면
- 비구조화된 시간

**영어 학습 적용:**
- 스피킹 연습은 저녁 이완 후에
- 스피치 전 5분 산책/스트레칭으로 DMN 활성화
- 떠오른 표현, 습관 → 즉시 메모 (판단은 나중에)

---

### 4. 멜라토닌 — 수면 = 학습 고정화

**역할:** 낮에 배운 내용을 장기기억으로 전환

**분비 조건:**
- 저녁 청색광 차단
- 취침 1시간 전 조명 50% 낮추기
- 규칙적 수면 시간

**영어 학습 적용:**
- 22시 이후 스마트폰 나이트 모드 필수
- 회고/일기는 종이로
- 수면 7-8시간 확보 = 내일 학습 효율의 기반

---

| 시간대 | 핵심 물질 | 최적 활동 | 구체적 학습 |

| 08:30-09:00 | 세로토닌 | 가벼운 준비 | 스트레칭 + 아침 식사 (계란, 바나나) |

## 핵심 실천 체크리스트

### 매일 필수

- [ ] 08:00 기상 후 30분 내 햇빛 노출
- [ ] 오전(09-11시)에 가장 어려운 학습 (단어/RC)
- [ ] LC 학습은 25분 포모도로 단위로 쪼개기
- [ ] 각 세트 완료마다 체크 표시
- [ ] 22시 이후 청색광 차단 (나이트 모드)
- [ ] 23시 회고는 종이로
- [ ] 7-8시간 수면 확보

### 주의사항

| 함정 | 현실 | 대안 |
|------|------|------|
| 긴 학습 블록 | 도파민 드롭 → 집중력 저하 | 25분 단위로 쪼개기 |
| 늦은 기상 | 세로토닌 충전 놓침 | 8시 기상 고정 |
| 밤늦게 스마트폰 | 멜라토닌 억제 → 수면 질 저하 | 22시 이후 종이/녹음기 사용 |
| 휴식 없이 몰아치기 | 번아웃 | 50분 작업 + 10분 휴식 |

> [!quote] 기억할 것
> 최적화는 "더 오래 공부하는 기술"이 아니라,
> **"뇌가 잘 작동하는 시간에 맞는 공부를 배치하는 기술"**이다.
