# 주말 10시간 공부 챌린지

## 개요

| 항목 | 내용 |
|------|------|
| 목표 | 매주 토·일 합계 10시간 이상 |
| 인원 | 4명 |
| 기간 | 2026.01.24 ~ 02.22 (4주) |
| 벌금 | 미달성 시 주당 1만원 |
| 정산 | 벌금을 성공자가 균등 분배 |

---

## 운영 규칙

### 인증 방법
- 타이머 앱(열품타) 또는 타임랩스 사용
- 공부 종료 시 단체 채팅방에 인증
- 마감: 매주 월요일 06:00까지 총 시간 제출

**시간대**
- 한국: 금요일 21:00 ~ 월요일 05:59
- 캐나다(Jane): 토요일 06:00 ~ 월요일 14:59

### 제외되는 시간
식사, SNS, 긴 휴식, 학습 무관 영상 시청

### 정산 방법
- 월요일 결과 집계 → 월요일 자정까지 송금
- 미송금 시 다음 주 벌금 2배
- 관리자 계좌: 농협 352-39407590-63

### 면제 규정
정말 피치 못할 사정으로 빠지게 될 경우 평일 10시간 보강 시 벌금 제외
1회 면제권 가능, 대신 면제권 사용시 성공하더라도 상금 분배 대상에서 제외.

## 분배 시나리오

| 실패 인원 | 성공 인원 | 총 벌금 | 1인당 수령액 | 비고 |
|-----------|-----------|---------|--------------|------|
| 0명 | 4명 | 0원 | 0원 | 전원 성공 |
| 1명 | 3명 | 10,000원 | 3,333원 | |
| 2명 | 2명 | 20,000원 | 10,000원 | |
| 3명 | 1명 | 30,000원 | 30,000원 | |
| 4명 | 0명 | 40,000원 | 다음 주 이월 | 전원 실패 시 누적 |

> [!note] 전원 실패 규칙
> 4명 모두 10시간 미달 시, 해당 주 벌금 40,000원은 다음 주로 이월되어 성공자 분배금에 합산됩니다.

### 계산 예시

```
1주차 결과
- A: 12시간 ○ → +3,333원
- B: 8시간 ✕ → -10,000원
- C: 11시간 ○ → +3,333원
- D: 10.5시간 ○ → +3,334원

총 벌금 10,000원 / 성공 3명 / 1인당 약 3,333원
```

## 주차별 기록

> [!info] 사용법
> 각 주차의 시간 값을 수정하면 달성 여부와 정산 금액이 자동 계산됩니다.

### 1주차

| 참가자 | 시간 |
|--------|------|
| A | (a1:: 0) |
| B | (b1:: 0) |
| C | (c1:: 0) |
| D | (d1:: 0) |

```dataviewjs
const page = dv.current();
const goal = 10;
const penalty = 10000;

const times = [page.a1, page.b1, page.c1, page.d1];
const names = ["A", "B", "C", "D"];

let results = times.map((t, i) => ({
	name: names[i],
	time: t,
	pass: t >= goal
}));

const failCount = results.filter(r => !r.pass).length;
const passCount = results.filter(r => r.pass).length;
const totalPenalty = failCount * penalty;
const reward = passCount > 0 ? Math.floor(totalPenalty / passCount) : 0;

dv.table(
	["참가자", "시간", "달성", "정산"],
	results.map(r => [
		r.name,
		r.time + "h",
		r.pass ? "○" : "✕",
		r.pass ? (reward > 0 ? "+" + reward.toLocaleString() + "원" : "-") : "-" + penalty.toLocaleString() + "원"
	])
);

dv.paragraph(`**집계**: 실패 ${failCount}명 / 성공 ${passCount}명 / 총 벌금 ${totalPenalty.toLocaleString()}원`);
```

## 4주 총합

```dataviewjs
const page = dv.current();
const goal = 10;
const penalty = 10000;
const names = ["A", "B", "C", "D"];

const allTimes = [
	[page.a1, page.a2, page.a3, page.a4],
	[page.b1, page.b2, page.b3, page.b4],
	[page.c1, page.c2, page.c3, page.c4],
	[page.d1, page.d2, page.d3, page.d4]
];

let summary = names.map((name, i) => {
	const times = allTimes[i];
	const total = times.reduce((a, b) => a + b, 0);
	const passWeeks = times.filter(t => t >= goal).length;
	const failWeeks = 4 - passWeeks;
	const paid = failWeeks * penalty;
	const received = times.map((t, week) => {
		if (t < goal) return 0;
		const weekTimes = [allTimes[0][week], allTimes[1][week], allTimes[2][week], allTimes[3][week]];
		const weekFails = weekTimes.filter(wt => wt < goal).length;
		const weekPasses = weekTimes.filter(wt => wt >= goal).length;
		return weekPasses > 0 ? Math.floor((weekFails * penalty) / weekPasses) : 0;
	}).reduce((a, b) => a + b, 0);
	const net = received - paid;
	return [name, total + "h", passWeeks + "주", (net >= 0 ? "+" : "") + net.toLocaleString() + "원"];
});

dv.table(["참가자", "총 시간", "달성", "순수익"], summary);
```
