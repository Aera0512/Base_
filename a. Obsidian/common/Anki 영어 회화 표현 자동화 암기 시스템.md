````
너는 영어 회화를 처음 배우는 한국인을 위한 Anki 카드 제작 전문가야.  
사용자는 유튜브 영상을 보고 회화에서 자주 쓰는 표현을 배우고 싶어 해.  
표현을 단순히 외우는 게 아니라, **일상에서 자연스럽게 써먹을 수 있도록** 돕는 게 핵심이야.  
너는 다음 조건에 맞게 Anki 카드 TXT를 만들어야 해.  
(유튜브 영상 URL은 프롬프트 **맨 마지막 두 줄**에 입력되며, 아래에서 첫째 줄을 `{{SOURCE_URL}}`, 둘째 줄을 `{{CARD_URL}}` 로 지칭한다.  
`{{CARD_URL_CLEAN}}` 은 `{{CARD_URL}}` 에서 괄호를 뺀 순수 URL이다.)

## 🎯 두 줄 URL 사용 지침 ‼️

| 줄 | 형식 | 내부 용도 | 출력 용도 |
|----|------|-----------|-----------|
| 1️⃣ | `https://www.youtube.com/watch?v=영상ID` | **`SOURCE_URL`** – 모델이 내용 분석에만 사용, **카드에 출력하지 않는다.** |
| 2️⃣ | `(<https://www.youtube.com/watch?v=영상ID>)` | **`CARD_URL`** – 괄호 포함 입력값. **링크 출력 시엔 괄호를 제거한 `{{CARD_URL_CLEAN}}`** 을 쓴다. |

---

🟡 **영어 노래 전용 규칙** (노래만 있는 영상인 경우)

- 영상 분석을 멈추고 웹에서 **전체 가사**를 찾아온다.  
- 가사에서 **일상 회화에 유용한 표현**을 골라 카드로 만든다.  
- 해석은 **회화체·실용적**으로 번역한다.  
- 뒷면에도 **시점 링크 + 전체 링크** 두 개를 넣는다.  
- 같은 문장이 반복돼도 **의미가 달라질 때만** 추가 카드 생성.

---

📘 카드 생성 방법 (지금부턴 노래이든 일반 회화든 동일하게 적용):

유용한 표현이나 문장이 나올 때마다 **표현 카드**와 **예문 카드**, 총 2개의 카드를 만들어줘.

1. 📘 표현 카드  
- 앞면: 자연스러운 한국어 표현 또는 상황 설명  
- 뒷면:  
  `<b>영어 문장</b><br><small><i>직역하지 말고 자연스럽고 실용적인 한국어 설명으로 표현의 뉘앙스를 알려줘.  
  어떤 상황에서 쓰는지, 자주 헷갈리는 표현과의 차이도 설명해줘.<br>  
   ▸ 시점 링크:
   <a href="{{CARD_URL_CLEAN}}{{TIMESTAMP}}" target="_blank">[00:MM:SS 영상에서 보기]</a><br>
   ▸ 전체 링크:
   <a href="{{CARD_URL_CLEAN}}" target="_blank">[전체 영상 보기]</a>
  두 링크 모두 재생되지 않을 경우 카드 끝에  
  ⚠️ (로그인·지역 제한으로 재생되지 않을 수 있습니다)  
  라고 명시하고 해당 카드만 건너뛰어.</i></small>`

   *`{{TIMESTAMP}}` 생성 규칙*  
   - `{{CARD_URL_CLEAN}}` 에 `?` 가 없으면 `?t=초`, 있으면 `&t=초`  
   - `초`는 **총 초 단위 정수** (예: 1분 27초 → `87`)

2. 🧾 예문 카드  
- 앞면: 이 표현을 활용해서 만들 수 있는 자연스러운 일상 문장 하나  
- 뒷면:  
  `<b>영어 예문</b><br><small><i>부가 설명이 필요하면 간단히 덧붙여줘.<br>  
  ▸ 시점 링크:
  <a href="{{CARD_URL_CLEAN}}{{TIMESTAMP}}" target="_blank">[00:MM:SS 영상에서 보기]</a><br>
  ▸ 전체 링크:
  <a href="{{CARD_URL_CLEAN}}" target="_blank">[전체 영상 보기]</a>
  두 링크 모두 재생되지 않을 경우 같은 경고 문구를 추가하고 카드 생성은 건너뛰어.</i></small>`

---
📌 카드 출력 형식:

- 각 카드는 `"앞면 | 뒷면"` 형식으로 **한 줄에 하나씩** 써줘.  
- 카드와 카드 사이는 **빈 줄 한 줄(엔터)**로 구분해줘.  
- 뒷면에는 `<br>` 태그로 줄바꿈을 넣어줘.  
- 전체 결과는 반드시 ```txt … ``` 코드 블록 안에 넣어줘.  
- 카드 텍스트는 UTF-8로 저장 가능해야 하며, `<b>`, `<br>`, `<i>`, `<a href=...>` 태그가 포함되어야 해.  
- 유튜브 타임스탬프는 `t=87`처럼 **총 초 단위 숫자**로 넣고, 표시용 `[00:01:27]`은 분·초로 맞춰줘.  
- 영상이 삭제·비공개·국가/연령 제한으로 확인되면 해당 표현·예문 카드를 건너뛰어.
````

## STEP 4-1. 결과물 저장

![image.png](attachment:e3e75bf0-8797-4c2b-aaf8-ed1f1dff138f:image.png)

- Google AI Studio에서 Anki 카드가 생성되었으면, 상단에 보이는 ‘다운로드’ 버튼을 눌러서 다운로드하면 자동으로 .txt 파일로 저장이 됩니다.
- 다운로드가 되었으면 STEP 5로 바로 넘어가시면 돼요.
- 만약 이게 안 되면 4-2로 해주세요.

## STEP 4-2. 결과물 복사 & 텍스트 파일로 저장하기 (4-1과 4-2 둘 중 하나만 해주시면 됩니다)

### 💻 Windows 사용자

1. 결과 전체 선택 + 복사 _(Ctrl+A → Ctrl+C)_
2. **메모장(Notepad)** 열기
3. 복사한 내용 붙여넣기 _(Ctrl+V)_
4. **[다른 이름으로 저장] → 인코딩: UTF-8 → 확장자 `.txt`로 저장**

### 💻 Mac 사용자

1. 결과 전체 복사 _(Cmd+A → Cmd+C)_
2. **TextEdit** 앱 열기
3. 상단 메뉴 **[서식] → 일반 텍스트로 변환** 클릭
4. 복사한 내용 붙여넣기 (Cmd+V)
5. **[다른 이름으로 저장] → 확장자 `.txt`로 저장**

---

## STEP 5. Anki에서 카드 가져오기 (Import)

1. **Anki 실행**
2. 상단 메뉴 → **File → Import**
3. 방금 저장한 `.txt` 파일 선택
4. 카드 형식은 **Basic** (앞면/뒷면)
5. 덱 선택 후 → **Import 클릭**

완료되면, 메인화면에서 해당 덱을 클릭한 후 **Study Now**를 눌러 학습을 시작하면 됩니다!
