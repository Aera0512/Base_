
여러 가지 중에서 나는 무엇을 선택해야하는 가

1. 외장 모니터 하나 제거하고 거치대 설치
   장점 : 돈 더이상 안 듬, 윈도우랑 맥간 연동 빠름.  / 지문인식 같은 것들 빠르게 터치 가능
   단점 : 맥북 듀얼 모니터이긴 한데, 작음 / 윈도우 쓸 때 화면 하나만 써야함. 
2. 디스플레이 링크
   장점 : 외장 모니터 2개 가능, 맥북 쓸 때 모니터 2개, 윈도우 쓸 때 2개, 모니터 간편하게 바꿔주고 사용 가능.
3. 모니터 암 하나 더 장만
   장점 : 맥북 쓸 때 모니터 2개 윈도우 쓸 때 2개, 누워있을 때, 모니터 간편하게 바꿔주고 사용 가능. + 간지
   단점 : 공간 차지로 인해서 기타 위치 두기 애매, 문이랑 부딪힘, 설치 불편 
4. 디스플레이링크 + 암 하나 더
