## 문장의 기본 구조와 동사
1. 주어 + 동사
2. 주어 + 동사 + 보어
3. 주어 + 동사 + 목적어
4. 주어 + 동사 + 간접목적어 + 직접목적어
5. 주어 + 동사 + 목적어 + 보어

## 동사를 통해 드러나는 시제
1. 현재시제가 나타나는 때와 의미
2. 과거시제와 미래
3. 현재진행형이 나타내는 때와 의미
4. 과거진행형과 미래진행형
5. 현재완료형의 개념과 의미
## 3. 동사에 의미를 더하는 조동사
1. can / may
2. must / should
3. will / would / used to
4. 조동사 + have p.p.

## 4. 주어와 동사의 관계를 보여주는 태
1. 주어가 동작을 받는 표현, be p.p.
2. 시제 형태에 주의할 수동태
3. 주의해야 할 수동태
4. 주어 + be p.p. + 목적어
5. 주어 + be p.p. + 명사/형용사 보어
