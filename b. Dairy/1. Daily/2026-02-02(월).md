> [!check] 📊 오늘의 달성률
> ```dataviewjs
> const content = await dv.io.load(dv.current().file.path);
> // DAY PLANNER 섹션만 추출
> const dayPlannerMatch = content.match(/## DAY PLANNER([\s\S]*?)(?=\n##|$)/);
> const dayPlannerSection = dayPlannerMatch ? dayPlannerMatch[1] : '';
> const total = (dayPlannerSection.match(/- \[.\]/g) || []).length;
> const done = (dayPlannerSection.match(/- \[x\]/g) || []).length;
> const pct = total > 0 ? Math.round((done / total) * 100) : 0;
> const bar = total > 0
>   ? ('█'.repeat(Math.round(pct / 5)) + '░'.repeat(20 - Math.round(pct / 5)))
>   : '░'.repeat(20);
> dv.paragraph(`**${done} / ${total}** 완료 · **${pct}%** \`${bar}\``);
> ```
