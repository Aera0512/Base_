#생산성 #자기계발 #Ai 
## 📋 개요

AI 도구들을 활용하여 **정보 수집 → 추출 → 연결**의 3단계로 구성된 개인 지식 관리 시스템을 구축하는 방법을 정리한 문서입니다.

---

## 🎯 핵심 워크플로우

---

## 1️⃣ 정보 수집 단계

### 🔍 주요 도구들

#### Perplexity Pro (`$20/월`)

- **핵심 기능**: 실시간 웹 검색 + 출처 명시
- **강점**: GPT-4o, Claude 3.5 Sonnet 모델 선택 가능
- **용도**: 최신 정보 및 뉴스/블로그 교차 검증
- **팁**: 모든 답변에 신뢰할 수 있는 출처 링크 제공

#### Consensus + SciSpace

- **핵심 기능**: 학술 논문 기반 메타 분석
- **강점**: 학계 동의율(Consensus Meter) 제공
- **용도**: 과학적 근거가 필요한 연구 질문
- **예시**: _"우유가 뇌 건강에 미치는 영향"_ → 수백 편 논문 분석 결과

#### Readwise Reader (`$9.99/월`)

- **핵심 기능**: 통합 콘텐츠 수집함 (Universal Inbox)
- **지원 형식**: 뉴스레터, 웹 기사, PDF, 유튜브 스크립트
- **AI 기능**: Ghostreader로 실시간 요약/질문
- **연동**: 옵시디언 자동 동기화 지원

## 2️⃣ 정보 추출 단계

### 🧠 핵심 도구: Google AI Studio (Gemini 1.5 Pro)

#### 주요 특징

- **컨텍스트 윈도우**: 200만 토큰 (책 수십 권 분량)
- **멀티모달**: 텍스트 + 이미지 + 비디오 동시 분석
- **비용**: **완전 무료** (중요한 장점)

#### 활용 사례

```
프롬프트 예시:"이 책 전체와 이 강의 시리즈 10개를 비교 분석하여, '뇌과학' 분야에서 새롭게 등장한 개념들만 추출해서 옵시디언 원자적 노트 형식으로 정리해줘"
```

## 3️⃣ 지식 연결 및 통합 단계

### 🔗 핵심 도구들

#### Claude 3.5 Sonnet (Desktop + MCP)

- **강점**: 최고 수준의 논리적 추론 능력
- **MCP 기능**: 옵시디언 폴더 직접 읽기/수정 가능
- **워크플로우**: 기존 노트와 신규 정보 간 논리적 연결

#### Smart Connections V2 (옵시디언 플러그인)

- **기능**: 로컬 벡터 검색 + 자동 연결 제안
- **장점**: 실시간 유사 노트 감지 및 연결 제안
- **모드**: 유료 API vs 로컬 Ollama (무료)

## 🚀 마스터 워크플로우 (4단계)

### 1단계: Capture (수집)

- **도구**: Perplexity + Consensus
- **결과물**: 신뢰할 수 있는 출처 리스트
- **저장위치**: Readwise Reader

### 2단계: Extract (추출)

- **도구**: Google AI Studio (Gemini 1.5 Pro)
- **명령어**: "내 옵시디언 템플릿에 맞춰 원자적 지식으로 변환"
- **결과물**: 구조화된 마크다운 노트

### 3단계: Synthesize (통합)

- **도구**: Claude 3.5 Sonnet + MCP
- **기능**: 기존 노트와 신규 정보 논리적 결합
- **결과물**: 업데이트된 지식 체계

### 4단계: Audit (검토)

- **작업**: 개인의 언어로 최종 수정
- **목적**: 장기 기억 정착
- **중요성**: ⭐⭐⭐⭐⭐ (가장 중요한 단계)

## 🆓 무료 버전 구성

### 도구별 무료 한계점 및 활용법

#### Perplexity AI (무료)

- **한계**: Pro Search 하루 5회
- **활용법**: 복잡한 질문에만 Pro Search 사용
- **무제한**: 일반 검색 기능

#### Consensus (무료)

- **한계**: 월 20개 AI 검색 크레딧
- **활용법**: 핵심 질문에만 사용
- **대안**: 수동으로 논문 검색 후 Gemini에서 분석

#### Google AI Studio ⭐

- **장점**: 200만 토큰 완전 무료
- **활용**: PDF 다중 업로드, 유튜브 링크 분석
- **접근**: `aistudio.google.com` (일반 Gemini 앱과 다름)

#### Claude.ai (무료)

- **한계**: 매우 적은 대화 횟수
- **활용법**: 최종 검토 및 논리 점검에만 사용

#### Smart Connections (로컬 모드)

- **대안**: Ollama 설치로 무료 무제한 사용
- **조건**: 어느 정도의 PC 사양 필요

## 🔄 무료 실전 워크플로우 (7단계)

### Step 1-2: 수집 및 확보

1. **Perplexity**에서 주제별 링크 3-5개 수집
2. 유튜브 스크립트 추출, 웹페이지 PDF 저장

### Step 3-4: 분석 및 명령

3. **Google AI Studio**에 모든 자료 업로드
4. "핵심 개념 5가지를 마크다운으로 정리" 명령

### Step 5-6: 정제 및 저장

5. **Claude**에서 비판적 분석 및 구조 개선
6. 완성된 마크다운을 옵시디언에 저장

### Step 7: 연결

7. **Smart Connections**로 관련 노트 자동 연결

## ⚠️ 주의사항 및 한계

### 정확성 관련

- **AI 환각**: 모든 AI 도구는 잘못된 정보를 생성할 수 있음
- **출처 확인**: 특히 과학적 사실은 원본 논문 직접 확인 필요
- **교차 검증**: 여러 도구의 결과를 비교 분석

### 비용 관련

- **가격 변동**: 언급된 가격은 2026년 1월 기준
- **사용량 고려**: 개인의 사용 패턴에 맞는 플랜 선택
- **무료 우선**: 먼저 무료 도구로 워크플로우 검증 후 유료 전환
