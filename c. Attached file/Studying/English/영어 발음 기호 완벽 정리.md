# 영어 발음 기호 완벽 정리

> 영어는 철자대로 소리나지 않는다. 발음 기호를 알아야 정확한 발음을 알 수 있다.

---

## Part 1. 개념 설명

### 모음 (Vowels) - 22개

#### 1. 단모음 vs 장모음(굴절모음)

**굴절모음**: 음이 꺾이는 모음. 위에서 아래로 소리가 굴곡을 그리며 내려간다.
**단모음**: 짧고 강하게 한 번에 터지는 소리. 굴곡 없이 일직선.

| 기호 | 이름 | 발음법 | 특징 |
|:---:|:---:|---|---|
| **iː** | 장모음 i | "이↗이↘" 올렸다 내리기 | 입술 양옆으로 쫙 벌림 (팔자주름) |
| **ɪ** | 단모음 I | "이"와 "에" 중간, 짧고 강하게 | 턱 거의 닫음 |
| **e** | 단모음 e | 한글 "애"보다 강하게 | 굴곡 없음 |
| **æ** | 나비발음 | "애↗애↘" 토하듯 | 턱 많이 내려감 |
| **ɑː** | 장모음 a | "아↗아↘" 깨달을 때 "아~" | 턱 많이 내려감 |
| **ɔː** | 장모음 ɔ | "어↗어↘" 아랫입술 그대로, 턱만 떨어뜨림 | |
| **uː** | 장모음 u | "우↗우↘" 입술 뾰족하게 모음 | 실제로 뽀뽀 입술 |
| **ʊ** | 말발굽발음 | "우"보다 "위"에 가까움, 강하게 | 입술 앞으로 살짝, 모으지 않음 |
| **ə** | 슈아(schwa) | 멍때릴 때 "어..." 힘 뺀 소리 | 강세 없음 |
| **ʌ** | 사까발음 | 놀랄 때 "어!" 강한 소리 | 강세 있음 |
| **ɜːr** | 강세 R모음 | 혀 미리 들어올려 "얼" | 강세 있을 때 |
| **ər** | 약세 R모음 | 위와 같은 소리, 약하게 | 강세 없을 때 |

#### 2. 이중모음 (Diphthongs)

**특징**: 두 모음이 만나 하나가 됨. 앞이 높고 뒤가 낮음. **길게** 발음해야 함.

| 기호 | 발음법 | 예시 |
|:---:|---|---|
| **aɪ** | "아↗이↘" | ice, high, time |
| **eɪ** | "에↗이↘" | may, day, say |
| **ɔɪ** | "오↗이↘" (깊은 오) | boy, toy, oil |
| **aʊ** | "아↗우↘" 입술 모아짐 | how, now, out |
| **oʊ** | "오↗우↘" 감탄사 "오~우" | go, no, show |

#### 3. R로 끝나는 이중모음

| 기호 | 발음법 | 예시 |
|:---:|---|---|
| **ɪər** | "이↗얼↘" | ear, here |
| **eər** | "에↗얼↘" | air, care |
| **ɑːr** | "아↗얼↘" | car, far |
| **ɔːr** | "오↗얼↘" (깊은 오) | or, door |
| **ʊər** | "우(짧게)↗얼↘" | sure, poor |

### 자음 (Consonants) - 24개

#### 1. 파열음 (Plosives)
공기를 막았다가 터트리는 소리

| 무성음 | 유성음 | 조음 위치 | 한글 비교 |
|:---:|:---:|---|---|
| **p** | **b** | 양 입술 | ㅍ/ㅂ보다 훨씬 강함 |
| **t** | **d** | 치경 (윗니 뒤 1-2cm) | ㅌ/ㄷ보다 뒤에서 |
| **k** | **g** | 연구개 (혀 뒷부분) | ㅋ/ㄱ보다 강함 |

> **핵심**: 영어 파열음은 한글보다 훨씬 강한 호흡으로!

#### 2. 마찰음 (Fricatives)
공기가 좁은 틈을 통과하며 마찰하는 소리

| 무성음 | 유성음 | 조음법 |
|:---:|:---:|---|
| **f** | **v** | 윗니를 아랫입술에 **살짝** 대고 (깨물지 않음!) |
| **θ** (뻔데기) | **ð** (돼지꼬리) | 혀를 윗니에 **살짝** 대고 |
| **s** | **z** | 가스 새는 소리 "쓰~" |
| **ʃ** | **ʒ** | 조용히 "쉬~" 입술 살짝 내밀어 |
| **h** | - | 한글 ㅎ과 유사 |

> **θ vs s 구별**: think ≠ sink / θ는 혀가 윗니 사이로!

#### 3. 비음 (Nasals)
코로 공기가 나오며 울리는 소리. **길게** 발음!

| 기호 | 한글 | 핵심 |
|:---:|:---:|---|
| **m** | ㅁ | 한글보다 **훨씬 길게** |
| **n** | ㄴ | 한글보다 **훨씬 길게** |
| **ŋ** | ㅇ받침 | 한글보다 **훨씬 길게** |

#### 4. 파찰음 (Affricates)

| 무성음 | 유성음 | 발음 |
|:---:|:---:|---|
| **tʃ** | **dʒ** | 한글 ㅊ/ㅈ보다 강하게, 혀를 윗몸에 눌러서 |

#### 5. 접근음 (Approximants)

| 기호 | 발음법 |
|:---:|---|
| **w** | 입술 뽀뽀 모양 → 떨면서 소리 |
| **r** | w 상태에서 혀를 공중으로 들어올림 |
| **j** | "이"소리 내면서 혀를 입천장에 가까이 → 덜덜 떨림 |
| **l** | 혀끝을 치경에 **반드시** 접촉 |

## Part 2. 미니멀 발음표

### 모음 총정리

```
┌─────────────────────────────────────────────────────┐
│                    단모음 (짧고 강하게)                │
├──────┬──────┬──────┬──────┬──────┬──────┬──────────┤
│  ɪ   │  e   │  æ   │  ʌ   │  ʊ   │  ə   │          │
│ 이/에 │  애  │ 애애 │ 어!  │ 위   │ 어.. │          │
├─────────────────────────────────────────────────────┤
│                    장모음 (올렸다 내려)                │
├──────┬──────┬──────┬──────┬──────────────┬─────────┤
│  iː  │  ɑː  │  ɔː  │  uː  │     ɜːr     │   ər    │
│ 이~이 │ 아~아 │ 어~어 │ 우~우 │    얼(강)    │  얼(약) │
├─────────────────────────────────────────────────────┤
│                    이중모음 (길게!)                   │
├──────┬──────┬──────┬──────┬──────────────┬─────────┤
│  aɪ  │  eɪ  │  ɔɪ  │  aʊ  │     oʊ      │         │
│ 아→이 │ 에→이 │ 오→이 │ 아→우 │    오→우     │         │
├─────────────────────────────────────────────────────┤
│                    R 이중모음                        │
├──────┬──────┬──────┬──────┬──────────────┬─────────┤
│  ɪər │  eər │  ɑːr │  ɔːr │     ʊər     │         │
│ 이→얼 │ 에→얼 │ 아→얼 │ 오→얼 │    우→얼     │         │
└──────┴──────┴──────┴──────┴──────────────┴─────────┘
```

### 자음 총정리

```
┌─────────────────────────────────────────────────────┐
│     파열음 (터트림)    │ 무성: p  t  k  │ 유성: b  d  g │
├─────────────────────────────────────────────────────┤
│     마찰음 (스침)      │ 무성: f  θ  s  ʃ  h │ 유성: v  ð  z  ʒ │
├─────────────────────────────────────────────────────┤
│     비음 (코소리)      │      m   n   ŋ      │ 길게!         │
├─────────────────────────────────────────────────────┤
│     파찰음            │ 무성: tʃ │ 유성: dʒ │               │
├─────────────────────────────────────────────────────┤
│     접근음            │   w   r   j   l     │               │
└─────────────────────────────────────────────────────┘
```

### 핵심 포인트 요약

| 구분 | 핵심 |
|---|---|
| 장모음 | 올렸다↗ 내려↘ 굴곡 만들기 |
| 단모음 | 짧고 강하게 한 방에 |
| 이중모음 | **길게!** 두 소리 연결 |
| 영어 자음 | 한글보다 **훨씬 강하게** |
| 비음 m/n/ŋ | 한글보다 **훨씬 길게** |
| t/d | 혀가 **치경**(윗니 뒤 잇몸)에 |
| f/v | 윗니를 아랫입술에 **살짝** (깨물면 안됨) |
| θ/ð | 혀를 윗니에 **살짝** 대고 |
| r | 혀 공중에 들어올림 |
| l | 혀끝이 **반드시** 치경에 닿음 |

## Part 3. 실전 연습

### 🎯 Day 1-3: 모음 기초

#### 연습 1: 장모음 굴곡 만들기
```
iː  →  "이↗이↘"  →  eat, see, cheese
ɑː  →  "아↗아↘"  →  hot, father
uː  →  "우↗우↘"  →  food, use, ooze
```
**체크**: 팔자주름이 생기는가? 입술이 뾰족해지는가?

#### 연습 2: 단모음 vs 장모음 구별
```
ɪ (짧게) vs iː (굴곡)
  sit  vs  seat
  hit  vs  heat
  bit  vs  beat
```

#### 연습 3: æ 나비발음
```
æ  →  "애↗애↘" 토하듯
  bad, sad, apple, cat
```
**체크**: 턱이 충분히 내려갔는가?

---

### 🎯 Day 4-6: 이중모음

#### 연습 4: 이중모음은 길게!
```
aɪ  →  ice (아~이~스)     time (타~이~ㅁ)
eɪ  →  day (데~이~)       say (세~이~)
aʊ  →  how (하~우~)       out (아~우~ㅌ)
oʊ  →  go (고~우~)        no (노~우~)
ɔɪ  →  boy (보~이~)       toy (토~이~)
```
**핵심**: 절대 짧게 하지 말 것!

#### 연습 5: R 이중모음
```
ɪər  →  ear (이~얼)       here (히~얼)
eər  →  air (에~얼)       care (케~얼)
ɔːr  →  or (오~얼)        door (도~얼)
```
**체크**: 혀가 미리 들려있는가?

---

### 🎯 Day 7-9: 자음 - 한국인 약점 공략

#### 연습 6: f/v - 깨물지 마!
```
윗니 → 아랫입술에 "살짝" → 바람 새는 소리

f  →  fan, fine, fox
v  →  van, vine, vox

비교: fan (f) vs van (v)
```
**오류 체크**: 입술을 깨물고 있지 않은가?

#### 연습 7: θ/ð - 혀를 내밀어!
```
혀 → 윗니에 살짝 → 그 사이로 공기

θ  →  think, three, bath
ð  →  this, that, mother

⚠️ think ≠ sink (θ ≠ s)
```

#### 연습 8: r vs l
```
r: 입술 모으고 혀 공중으로 들어올림
   red, right, wrong

l: 혀끝이 치경에 반드시 닿음!
   light, love, feel

비교:
  right vs light
  wrong vs long
  read vs lead
```

### 🎯 Day 10-14: 통합 연습

#### 연습 9: 무성음 ↔ 유성음 세트
```
성대 안 떨림 → 성대 떨림
p → b :  pay → bay
t → d :  tie → die
k → g :  came → game
f → v :  fan → van
θ → ð :  thin → then
s → z :  sue → zoo
ʃ → ʒ :  션 → 쥬앙
tʃ → dʒ: choke → joke
```

#### 연습 10: 비음 길게 유지
```
m~~~~~  →  mom (마~~~~~ㅁ)
n~~~~~  →  nine (나~~~~~인)
ŋ~~~~~  →  sing (씽~~~~~)
```

#### 연습 11: 문장 연습
```
1. "She sells seashells" 
   → ʃ와 s 구별

2. "Think three thoughts"
   → θ 연속

3. "Really red lorry"
   → r과 l 구별

4. "Very fine view"
   → v와 f 연습
```

## 참고 자료

- 원본 영상: 소통영어 유황 - 영어 발음기호 읽는 법 (모음편, 자음편)
- 무성음 = 성대 안 떨림 / 유성음 = 성대 떨림
- 파열음: 공기를 막았다 터트림
- 마찰음: 좁은 틈으로 공기 통과
- 비음: 코로 소리 울림
