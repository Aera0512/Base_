## 📌 이번 성찰

### 1. 경험 떠올리기
> 설렜던, 행복했던, 짜릿했던, 몰입했던, 가슴이 꽉 찼던 경험

```
여기에 경험 적기:
1. Fame 공연을 보며, 그리고 노래를 부르며 사람들이 한 마음으로 뭉친 것을 보고 눈물이 차오른 경험
2. 일본 여행 갔을 때, 렌트카를 타고 자연을 바라보니 기분이 좋아졌던 경험.
3. 알바하면서 사람들과 선한 에너지를 주고 받고 만족스러웠던 경험.
4. 하루 종일 노래를 부르고 녹음하고 듣기를 반복하는데 시간이 가는 줄 몰랐던 경험.
5. 아크로바틱 체력 때 정신력으로 가장 먼저 완주를 해서 뿌듯했던 경험.
6. 내가 체계화한 정보를 바탕으로 세계를 이해했을 때 만족했던 경험.
7. 2학년 때 후배들한테 인기가 많아서 자신감이 차올랐던 경험.
8. 운동을 하고 몸을 멋있게 만든 후에 친구들한테 인정 받았던 경험.
9. 1학년 때 장학금을 받겠다는 목표를 세우고 정말 장학금을 받아서 용기를 얻은 경험.
10. 원하는 대학교에서 예상하지 못한 합격을 받아서 하루 종일 기분이 좋았던 경험.
11. 책 하나를 완독했을 때 뿌듯함을 느낀 경험.
12. 지금처럼 궁금한 것을 찾아 탐구하고 정리하는 때에 높은 집중력과 몰입감을 느낀 경험.
```

### 3. 2차 성찰

**이 상황이 나에게 주는 가치는?** (왜 좋았지?)
```
1. 페임 공연 : 연대감 + 아름다움 + 확신 -> 내가 하는 일이 정말 아름답구나
2. 일본 여행 : 아름다움 + 추구가치 + 활기 -> 새로운 아름다움의 발견
3. 알바 대화 : 추구가치 + 만족 + 이타심 -> 상대방을 행복하게 해주었다는 만족감
4. 노래 시간 : 몰입감 + 관심사 + 발전 -> 정보 체계화 및 신체 활동의 만족감 / 난이도 하, 감각
5. 아바 체력 : 성취 + 고통인내 + 증명 -> 사람들 앞에서 나의 노력으로 나의 능력을 증명
6. 학습 이해 : 호기심 + 몰입감 + 발전 -> 나의 노력으로 인과관계를 이해
7. 후배 인기 : 희망 + 뜻밖 + 증명 -> 내가 괜찮은 사람이라는 만족감
8. 운동 성장 : 성취 + 고통인내 + 인정 -> 사람들의 나의 능력을 인정함
9. 공부 달성 : 성취 + 고통인내 + 전율 + 계획 -> 높은 목표를 성취한 만족감
10. 대학 합격 : 인정 + 고통인내 + 희망 -> 생각보다 나 자신이 괜찮구나에 대한 만족감
11. 독서 완독 : 계획 + 성장 -> 지식 성장
12. 탐구 정리 : 이해 + 학습 + 체계 -> 학습과 확장
```

```

2. 자연의 아름다움
3. 인간 관계의 아름다움과 이타적 행동
4. 기존 지식을 활용한 실시간 피드백, 신체 활동의 만족감
5. 힘든 과정 끝의 목표 달성
6. 현상 체계화
7. 좋은 사람
8. 힘든 과정 끝의 성장
9. 힘든 과정 끝의 목표 달성
10. 힘든 과정 끝의 목표 달성
11. 지식의 성장
```
