# Inbox

This is the temporary storage and processing area for all incoming information.

## Purpose

- **Capture everything** first without worrying about organization
- **Process regularly** and move to appropriate CMDS categories
- **Keep it clean** - inbox should be empty or near-empty regularly

## Workflow

1. **Capture** - Add new notes here quickly
2. **Review** - Daily or weekly review
3. **Process** - Categorize into proper CMDS folders
4. **Archive** - Move processed items out of inbox

## Subdirectories

- `01. Daily Notes/` - Daily journal entries
- `02. Weekly Notes/` - Weekly reviews and planning
- `03. Claude Code/` - All code outputs from Claude
- `04. Excalidraw/` - Visual diagrams
- `05. Canvas/` - Canvas notes
- `06. GenAI Chats/` - AI conversation logs
- `07. Clippings/` - Web clippings and saved content
