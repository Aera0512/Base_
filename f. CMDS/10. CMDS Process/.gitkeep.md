# CMDS Process

Documentation of the CMDS process: Connect → Merge → Develop → Share

## Purpose

This folder contains notes about the CMDS process itself:
- How each phase works
- Best practices
- Workflow templates
- Process improvements

## The Four Phases

1. **Connect** (연결) - Idea discovery and connection
2. **Merge** (통합) - Literature and knowledge integration
3. **Develop** (개발) - Data collection and methodology application
4. **Share** (공유) - Creative work and output dissemination
